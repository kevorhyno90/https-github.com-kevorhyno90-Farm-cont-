Icon Generator for Windows

Because Windows sometimes fails to use SVG icons for desktop shortcuts, we must convert your logo to standard PNGs.

Instructions:
1. Right-click the LARGE image below and choose "Save image as..."
2. Save it exactly as icon-512.png inside your project's public folder (overwrite the old one).
3. Right-click the SMALL image below and choose "Save image as..."
4. Save it exactly as icon-192.png inside your project's public folder (overwrite the old one).
